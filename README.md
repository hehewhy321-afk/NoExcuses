### Prerequisites

### Setup
1. **Clone the repo**:
   ```bash
   git clone https://github.com/hehewhy321-afk/ai-roast-reminder.git
   cd ai-roast-reminder
   ```

2. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

3. **Configure Backend**:
   - Initialize Convex: `npx convex dev`
   - Set your API keys in the Admin panel within the app or directly in Convex.

4. **Run the App**:
   ```bash
   flutter run
   ```
